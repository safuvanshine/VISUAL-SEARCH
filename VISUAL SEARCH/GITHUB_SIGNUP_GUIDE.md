# How to Create a GitHub Account

Follow these simple steps to set up your free GitHub account. You will need this to put your jewelry app online as a website.

### Step 1: Visit the Signup Page
1.  Open your browser and go to: **[https://github.com/signup](https://github.com/signup)**
2.  You will see a "Welcome to GitHub! Let's begin the adventure" message.

### Step 2: Enter Your Email
1.  Type in your active email address.
2.  Click **Continue**.

### Step 3: Create a Password
1.  Choose a strong password (at least 8 characters).
2.  Click **Continue**.

### Step 4: Choose a Username
1.  Type in a name you'd like to use (e.g., `shine-jewelry` or `jewel-admin`).
2.  If the name is available, click **Continue**.
3.  **IMPORTANT**: Remember this username! You'll need to tell me what it is.

### Step 5: Email Preferences
1.  Type `y` if you want updates, or `n` if you don't.
2.  Click **Continue**.

### Step 6: Verify You Are Human
1.  Follow the simple puzzle instructions on the screen (e.g., rotate an image).
2.  Click **Create account**.

### Step 7: Enter the Launch Code
1.  GitHub will send an 8-digit code to your email.
2.  Open your email, copy the code, and type it into the GitHub page.

### Step 8: Final Questions (Optional)
1.  You can skip these questions or just click through them.
2.  Choose **Free** for the plan (you don't need to pay for anything).

---

**Once you see your Dashboard (the home screen of GitHub), you are finished!**
Just come back here and send me your **Username**.
